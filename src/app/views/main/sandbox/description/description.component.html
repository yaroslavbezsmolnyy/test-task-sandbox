Test task includes 2 levels of complexity: base and additional (*)

Base level: You can ignore the time in array intakes. For example, even if a function runs at 8 pm, we still calculate this day as a full day of intake.

Additional *: To implement function taking into account time in the array intakes. For example, if we run a function at 1 pm, a person will miss morning intake ({{ '{' }}time: "9:15", pills: 1}) he/she will have 1 pill in stocks remain, which he’ll intake in his last day of intake.
